# 位操作

位操作符总是将数字转换为对应二进制再进行操作


### >> (右移)
```js
const x = 8
x >> 1 // 1000 >> 1 = 0100(2) = 4
x >> 2 // 1000 >> 2 = 0010(2) = 2
x >> 3 // 1000 >> 3 = 0001(2) = 1
```

### << (左移)
```js
const x = 1 // 0001(2)
x << 1 // 0001 << 1 = 0010(2) = 2
x << 2 // 0001 << 2 = 0100(2) = 4
x << 3 // 0001 << 3 = 1000(2) = 8
```

### | (按位或)
```js
const x = 17 // 17 = 16 + 1
x | 16 // 17
x | 8 // 16 + 1 + 8 = 25
x | 4 // 16 + 1 + 4 = 21
x | 5 // 5 = 4 + 1; 16 + 1 + 4 = 21
```

### & (按位与)
```js
const x = 17 // 17 = 16 + 1
x & 16 // 16
x & 8 // 0
x & 4 // 0
x & 5 // 1
```


## 取整

```js
console.log(123.456 | 0) // 123
```
